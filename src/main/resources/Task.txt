Итерация 2
Реализовать сохранение состояния банкомата. Использовать ObjectMapper(или любой другой вариант) для сохранения списка людей в банкомате.
Часть 1 (без авторизации).
Добавить класс Person(name, balance, cardNumber). При запуске приложения загружать json со списком людей, по имени находить пользователя (пока без логина/пароля)
и использовать его баланс и номер карты для переводов другим пользователям. Если пользователь не нашелся по имени – создавать нового, генерировать номер карты(на твое усмотрение)
и сохранять его в список уже существующих. Доработать операцию с отправкой денег, чтобы можно было указывать номер карты для перевода. После выхода из приложения сохранять измененный
список пользователей.
Желательно вынести логику операций в методы класса Main, чтобы увеличить читаемость кода.
Часть 2.
Доработать модель пользователя таким образом, чтобы можно было сохранять логин пароль и добавить регистрацию/авторизацию при входе.
Дополнительно
1. Расширить модель и функциональность, добавив вывод истории операций
2. Добавить возможность выходить из аккаунта пользователя, не выходя из всего приложения, чтобы можно было зайти за другого. Т.е. реализовать logout
